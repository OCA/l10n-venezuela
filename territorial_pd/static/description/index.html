Municipalities and Parishes

For deeper political division of Country and State

All federal states, municipalities and parishes of Venezuela

  • 24 federal states
  • 335 municipalities
  • 1136 parishes

Get more done in less time

It's integrated with the base models res.country
and res.country.state

  • res.country
  • res.country.state
  • res.country.state.municipality
  • res.country.base.municipality.parish

Each model contains the char fields of name and code of each Municipality and Parish

Relational fields:

  • res.country.state: 
    • municipality_ids --> id of the state's municipalities
  • res.country.state.municipality
    • state_id --> id of the State to which the municipality belongs
    • parish_ids --> id of the municilapity's parishes
  • res.country.state.municipality.parish
    • municipality_id --> id of the Municipality to which the parish belongs

UPGRADE:

This module also contains the nationalities associated with each country (for the Americas, Asia and Europe continents)
