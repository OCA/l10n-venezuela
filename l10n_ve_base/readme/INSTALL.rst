To install the modules, you must follow the following module installation order:

1) territorial_pd

2) l10n_ve_base

3) unique_vat_by_partner

4) vat_type_partner

5) l10n_ve_withholding

6) l10n_ve_vat_ledger

7) l10n_ve_invoice_reports
